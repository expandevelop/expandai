# Pesquisa de redesign do front-end da ExpandAI

## Referência de marca e acesso aos sites

A investigação inicial mostrou que o domínio informado pelo usuário, `iaexpand.com.br`, não respondeu por DNS no ambiente atual. Em busca pública, foi localizado o domínio `https://www.iaexpand.com/`, com título **"I. A. Expand LLC (Innovate, Automate, Expand) | Transform Your Business Now"**, que passa a ser a referência pública mais próxima disponível nesta sessão.

A conclusão provisória é que a nova interface deve seguir uma linha de marca **azul + verde**, com linguagem mais limpa, menos densa e com melhor hierarquia visual, mas ainda será necessário consolidar a paleta exata e confrontá-la com a estrutura atual do projeto web antes da implementação.

## Referência visual confirmada do site correto

A inspeção técnica do site `https://aiexpand.com.br/` confirmou uma linguagem visual bastante enxuta, com mensagem central forte e uso predominante de **azul escuro profundo** e **verde vivo**, apoiados por neutros claros. Os principais códigos observados no HTML foram:

| Papel visual | Cor observada |
| --- | --- |
| Azul base profundo | `#0f1d2f` |
| Azul secundário | `#1e3a5f` |
| Verde principal | `#16a34a` |
| Verde de destaque | `#22c55e` |
| Branco | `#ffffff` |
| Cinza claro | `#f3f4f6` |
| Cinza médio | `#9ca3af` |
| Cinza textual | `#4b5563` |

O conteúdo textual público atual do site é minimalista e reforça uma proposta de marca limpa e tecnológica, centrada em expansão comercial inteligente. Isso confirma que o redesign da ExpandAI deve abandonar superfícies densas e adotar uma composição com mais respiro, hierarquia clara, contraste forte e blocos mais bem separados.
